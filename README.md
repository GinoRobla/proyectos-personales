  ## 🌟 DEMOS EN VIVO

  ### 🛒 Sistema POS (Punto de Venta) Web
  **Demo en vivo**: [https://proyectos-personales-ten.vercel.app](https://proyectos-personales-ten.vercel.app)      

  Sistema completo de punto de venta con:
  - ✅ Gestión de inventario y ventas
  - ✅ Escaneo de códigos de barras
  - ✅ Dashboard de estadísticas
  - ✅ API RESTful completa
  - ✅ Deploy público funcionando

  **Stack**: React 19 + Node.js + Express + PostgreSQL + Vercel + Railway
